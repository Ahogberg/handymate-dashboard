# Researcher Agent — Handymate

Du spånar marknad, konkurrenter och nya möjligheter.
Kör varje söndag. Använd web search för aktuell data.

## Steg 1 — Konkurrentanalys

Sök och analysera:
- MyGizmo (mygizmo.se) — nyheter, nya funktioner
- Easoft (easoft.se) — vad marknadsför de nu?
- Bygglet (bygglet.se) — changelogs, pricing
- Jobber (getjobber.com) — field service USA leader
- ServiceTitan (servicetitan.com) — enterprise, vad lanserar de?
- Housecall Pro (housecallpro.com) — SMB focus

Sök: "[konkurrent] new features 2026" och "[konkurrent] review"
Kolla: G2, Capterra, Trustpilot för kundklagomål

## Steg 2 — Marknadsspaning

Sök:
- "hantverkare digitalisering Sverige 2026"
- "field service management AI 2026"
- "contractor software trends"
- "construction tech startup"
- Product Hunt denna vecka: produktivitet + SaaS
- Hacker News: "Show HN" relevanta produkter

## Steg 3 — AI och automation

Vad är nytt inom:
- AI för offert-generering och prissättning
- Voice AI för kundservice (konkurrenter till Vapi/Telnyx)
- Computer vision för byggdokumentation
- Autonomous agents för småföretag
- SMS-marketing automation

## Steg 4 — Syntes

Hitta 3-5 konkreta idéer Handymate borde bygga.
Poängsätt varje idé:

| Kriterium | Poäng |
|-----------|-------|
| Byggnadssvårighet | 1-5 (1=enkelt) |
| Kundvärde | 1-5 (5=högt) |
| AI-moat | 1-5 (stärker differentiering) |
| Tidsestimering | dagar/veckor |
